# WAKFU Sacrier 技能策划参考

> 来源：WAKFU 官方百科 Sacrier 职业页面。抓取日期：2026-06-25。

本文档整理为游戏开发策划参考格式。数值取自官方等级选择器的 Lv1、Lv50、Lv100、Lv150、Lv200、Lv245。若源页面没有暴击效果，暴击列保留为 `-`。

## 策划说明

- 元素伤害和护甲数值会随技能等级成长；表格只列代表性等级节点，不展开全部 245 个中间等级。
- 位移、稳定、HP 百分比、WP/MP 改动、被动触发条件等固定效果按源页面含义翻译整理。
- `Flame Return`、`Fury`、`Armor`、`Lock`、`Dodge`、`Punishment` 是 WAKFU 原机制；落地到本项目时需要映射成本项目机制。
